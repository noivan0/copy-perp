|---|------|---------|------|

| 5 | **주소 표시** | 연결 후 지갑 주소(`xxxx...xxxx`) 표시 | ☐ PASS / ☐ FAIL |

## 실패 시 즉시 확인 항목

| 증상 | 원인 후보 |
|------|---------|
| 페이지 안 뜸 | 프론트 빌드/배포 오류 |
| Privy 팝업 안 뜸 | `NEXT_PUBLIC_PRIVY_APP_ID` 환경변수 미설정 |
| 지갑 연결 실패 | Privy 대시보드 도메인 허용 목록 미등록 |
| 리더보드 비어있음 | 백엔드 API CORS or `/traders` 응답 오류 |
| 팔로우 실패 | `POST /follow` 400 → 백엔드 로그 확인 |
| 자물쇠 없음 | G14 HTTPS 미설정 — Dev팀장 긴급 처리 |
