- **현재 상태:** API 서명 구조 정상, 엔드포인트 응답 정상, approve만 pending

- 해커톤 데모에서는 유효한 testnet 주소로 교체하거나 MockFollower만 사용 권장

- 실서비스 배포 시: 기존 서버 종료 후 `supervisord -c supervisord.conf` 단독 실행 필요
