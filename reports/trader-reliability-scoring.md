## 신뢰도 점수 계산 체계 (5개 축)

### 1. 수익성 축 (Profitability) — 30점

```
Profit Factor = 총이익 / 총손실
PF < 1.0  → 0점 (손실 구간)
PF 1.0~1.5 → 10점
PF 1.5~2.0 → 20점
PF > 2.0   → 30점
```

### 2. 일관성 축 (Consistency) — 25점

```
Sharpe Ratio (거래별)
< 0      → 0점
0~0.1    → 8점
0.1~0.3  → 16점
> 0.3    → 25점

+ 연속 손실 패널티
연속 손실 5건 이상: -5점
연속 손실 10건 이상: -15점
```

### 3. 리스크 관리 축 (Risk Control) — 25점

```
최대 드로우다운 (MaxDD)
> 50%    → 0점
30~50%   → 8점
10~30%   → 16점
< 10%    → 25점

청산(liquidation) 이력: 1회당 -10점
```

### 4. 표본 신뢰도 축 (Sample Validity) — 10점

```
거래 수 (Trade Count)
< 20건   → 0점  (통계적으로 무의미)
20~50건  → 5점
50~100건 → 8점
> 100건  → 10점

+ 기간 보정
분석 기간 < 7일 → 점수 × 0.5
분석 기간 7~30일 → 점수 × 0.8
분석 기간 > 30일 → 점수 × 1.0
```

### 5. 스타일 안정성 축 (Style Stability) — 10점

```
Kelly Criterion
< 0 → 0점 (음의 기대값 — 반드시 제외)
0~0.05 → 5점
> 0.05 → 10점

+ 리스크/리워드(RR) 패널티
RR < 0.5 → 승률이 높아도 큰 손실 한 방에 날아감 → -5점
```

## 종합 판정 기준

```
80~100점 → ✅ TIER A: 팔로우 추천 (copy_ratio 기본값 높게)
60~79점  → ⚠️ TIER B: 조건부 팔로우 (copy_ratio 낮게)
40~59점  → 🔶 TIER C: 소액 테스트만 허용
0~39점   → ❌ TIER D: 팔로우 차단
```

## 주요 함정 패턴 (블랙리스트 룰)

### 1. "순간 고ROI" 함정
- 분석 기간이 짧고 ROI가 비정상적으로 높음
- ex) 7일 만에 1000% → 단일 대박 or 운
- 대응: **기간 보정 계수** 적용

### 2. "마르팅게일" 함정
- MaxDD가 자본 100% 이상
- 손실을 더 큰 베팅으로 만회하는 패턴
- 대응: **MaxDD > 50% 즉시 차단**

### 3. "승률 위조" 함정
- 승률은 80~100%이지만 Kelly < 0
- 작은 이익 수십 번 + 대형 손실 1~2번
- ex) 4TYEjn9P: 승률 80%지만 PF=1.09, MaxDD=121%
- 대응: **PF < 1.3이면 승률 무시**

### 4. "공유 포지션" 함정
- 동일 시각 여러 트레이더가 같은 포지션
- ex) HtC4WT6J, E8j5xSbG가 동일 BTC 청산
- 의미: 독립적 알파가 없음 (서로 마켓메이킹 상대방)
- 대응: **포지션 상관관계 < 0.3 요구**

## 노이반님 언급 지표와의 연동

| 지표 | 신뢰도 점수 연동 방법 |
|------|-------------------|
| RSI | 과열 구간(>80, <20) 진입 체결은 신뢰도 페널티 |
| ADX + regime | trend 구간 체결만 카운트 (range에서 방향성 배팅은 노이즈) |
| BB pos ratio | 0.9+ (상단 돌파) 진입 → 고위험 태그 |
| score_base vs score_with_of | OF 보정 후 점수 향상 → 실제 모멘텀 확인된 체결만 신뢰 |
| Supertrend | 체결 방향 ≠ Supertrend 방향이면 역추세 → 신뢰도 할인 |
| of_dir / of_imb | OF 방향과 체결 방향 일치 여부 → 일치시 +, 불일치시 - |

## 구현 우선순위

**Phase 1 (지금 당장):**
- Profit Factor + MaxDD + Kelly로 TIER D 자동 차단
- 리더보드에서 PF < 1.0 또는 MaxDD > 100% 트레이더 제거

**Phase 2 (1주 내):**
- 거래별 Sharpe, 연속 손실 추적
- 신뢰도 점수 대시보드 (팔로워가 볼 수 있게)

**Phase 3 (데모 전):**
- RSI/ADX/OF 지표 연동 → 체결 품질 점수화
- "이 트레이더의 최근 거래 중 몇 %가 고품질 시그널?" 표시
